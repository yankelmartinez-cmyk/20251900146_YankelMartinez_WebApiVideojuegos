# WebApi Videojuegos

Proyecto hecho para la clase de Programación Orientada a Objetos usando ASP.NET Core Web API con C#.
Hice una API de videojuegos populares en Honduras con las operaciones básicas CRUD.

## Endpoints

Para probarlos en Postman, el puerto puede variar según tu Visual Studio.

- GET (Listar todos los videojuegos): https://localhost:{tuPuerto}/api/VideojuegosController/ListarVideojuegos
- GET (Obtener videojuego por ID): https://localhost:{tuPuerto}/api/VideojuegosController/ObtenerVideojuegoPorId/{id}
- POST (Agregar videojuego): https://localhost:{tuPuerto}/api/VideojuegosController/AgregarVideojuego
- PUT (Actualizar videojuego): https://localhost:{tuPuerto}/api/VideojuegosController/ActualizarVideojuego
- DELETE (Eliminar videojuego): https://localhost:{tuPuerto}/api/VideojuegosController/EliminarVideojuego/{id}
